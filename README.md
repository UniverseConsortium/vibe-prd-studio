# Vibe PRD Studio

> English page. 点击这里查看中文文档：**[简体中文](./README.zh-CN.md)**

A Next.js app that helps Vibe Coding beginners turn fuzzy ideas into professional PRD documents.

## Quick Start

```bash
npm install
npm run dev
```

Open <http://localhost:3000>

## Build

```bash
npm run build
npm run start
```

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- ESLint
